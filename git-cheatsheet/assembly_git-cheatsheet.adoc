= Git Cheat Sheet

This humble resource introduces basic Git concepts and helps to install Git on your computer.

== Prerequisites
* A working computer or laptor with any operating system.
* An Internet connection.
* A command line application, for example Terminal on Linux and macOS or Windows Terminal on Windows.
